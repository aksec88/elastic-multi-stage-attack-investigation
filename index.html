04 // DETECTION ENGINEERING

Custom Detection Rules

SSH Brute Force Detection Medium
Threshold Rule · ≥5 failed authentication attempts in 5 minutes from a single source IP
system.auth.ssh.event:failed AND user.name:* GROUP BY source.ip HAVING count ≥ 5 OVER 5m
4,000+ Alerts fired
RDP Brute Force Detection Medium
Threshold Rule · Windows Event ID 4625 · ≥5 failed logon attempts in 5 minutes
event.code:4625 AND user.name:Administrator GROUP BY source.ip HAVING count ≥ 5 OVER 5m
272 Alerts fired
Mythic C2 Apollo Agent — Process Masquerade Critical
Custom Query Rule · Sysmon Event ID 1 (Process Creation) · OriginalFileName match
event.code:1 AND process.pe.original_file_name:Apollo.exe AND NOT process.name:Apollo.exe
4 Alerts fired
Elastic Defend — Endpoint Security Events Variable
EDR Policy Rule · Malware prevention + memory threat detection · Windows Server 2022
event.category:malware AND event.action:(quarantine OR prevention) AND agent.type:elastic_defend
2+ Alerts fired
05 // INCIDENT RESPONSE

7-Step Investigation Framework

01 Verification Validate source IP/user against known baselines. Review historical access patterns. Assess full context of flagged activity before escalating.
02 Threat Intelligence IP reputation lookup, classification, and tagging. Hash analysis for known malicious binaries. Correlate against external threat feeds.
AbuseIPDB GreyNoise
03 Scope Assessment Identify all affected systems and users. Determine if activity is isolated or part of a wider campaign. Pivot across related events in Kibana Discover.
04 Success Rate Analysis Determine whether the attack succeeded. Identify privilege escalation attempts or lateral movement paths. Map Logon IDs to post-authentication actions.
05 Post-Attack Activity If successful: trace file access, data exfiltration, and persistence mechanisms. Use Process GUID correlation (Sysmon EID 1 + 3) to reconstruct full timelines.
06 Containment & Remediation Trigger host isolation via Elastic Defend EDR. Credential rotation. Network-level blocks. Firewall rule updates. Validate controls are effective.
Elastic Defend Host Isolation
07 Documentation & Closure Structured investigation notes. Full event timeline. Extracted IOCs. Lessons identified. osTicket case closed with audit trail preserved for compliance.
osTicket Webhook Auto
